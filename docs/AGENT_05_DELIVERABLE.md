## Personality-Driven Animations

### Animation Budget (Mobile)
- **Victory**: 3â€“5 second loop
- **Defeat**: 2â€“3 second clip
- **Boost**: 0.5â€“1.0 second burst
- **Hit**: 0.5 second reaction

### Emotional Beats
Each character has distinct reactions to:
1. **Getting 1st place**: Triumphant
2. **Falling to last**: Frustrated
3. **Perfect drift**: Satisfied
4. **Near-miss dodge**: Relieved/excited

### Accessibility
- **Subtitles**: All voice lines captioned
- **Reduced Motion Mode**: Tones down excessive animations
- **Emoji Indicators**: Visual emotion cues for audio-impaired players

- **Core Lines**: 20 (start, boost, hit, victory, defeat, item use)

- **Voice**: Always clear, ducked during intense moments

## Keep All Characters Viable

### Balance Principles
1. **No character has >55% win rate** in skilled play
2. **No passive ability accounts for >10% performance swing**
3. **Every character viable on every track**
4. **Preference = playstyle, not power**

### Testing Validation
- [ ] Character win rates within Â±5% across 1000+ races
- [ ] No single character dominates leaderboards (max 15% representation)
- [ ] Player surveys show "I like my character" >80%
